⚖️ Scraper de expedientes — PJN

Descarga automática de causas del Sistema de Consulta Web (SCW)
del Poder Judicial de la Nación

Python 3 Playwright scw.pjn.gov.ar

Herramienta de línea de comandos que automatiza la descarga de expedientes desde el Sistema de Consulta Web (SCW) del Poder Judicial de la Nación. Abre un navegador, hace una pausa para que ingreses el apellido en Parte y resuelvas el captcha a mano, y luego recorre la tabla de resultados: entra a cada causa, registra todas las actuaciones en un CSV y descarga los adjuntos (PDFs), organizados en carpetas por causa.

⬇️ Descargar el programa (ZIP) 📄 Descargar guía (PDF) Ver el código Repositorio
⚠️ Importante — esto se ejecuta en tu computadora. Por seguridad, los navegadores prohíben que una página web lea el contenido de otra (la web del Poder Judicial). Por eso la descarga de expedientes no puede correr dentro de esta página ni en un servidor: el programa abre un navegador en tu máquina para que vos resuelvas el captcha y luego baja todo solo. Esta página te da todo lo necesario para hacerlo en 5 minutos. 👇

🚀 Guía rápida

1. Instalar Python (una vez)

Descargalo de python.org/downloads y al instalar tildá “Add Python to PATH”.

2. Bajar el programa (una vez)

Usá el botón “⬇️ Descargar el programa (ZIP)” de arriba y descomprimí la carpeta Judicial.

3. Abrir la terminal en esa carpeta

Entrá a la carpeta, escribí cmd en la barra de dirección y apretá Enter (se abre una ventana negra).

4. Preparar (una vez)

pip install playwright
playwright install chromium

5. Usarlo (cada vez)

python scraper_pjn.py

Se abre el navegador → poné el apellido, resolvé el captcha, apretá Buscar → volvé a la ventana negra y apretá Enter. Los PDFs se guardan en la carpeta expedientes_pjn.

¿No anda la primera vez?

python scraper_pjn.py --debug

Genera la carpeta debug_scw; enviá el archivo debug_scw/tablas.txt para terminar de ajustarlo.

Instalación

pip install -r requirements.txt
playwright install chromium

Uso

python scraper_pjn.py
  1. Se abre Chromium. Ingresá el apellido en el campo Parte, resolvé el captcha y hacé clic en BUSCAR.
  2. Cuando veas la tabla de resultados, volvé a la terminal y presioná ENTER.
  3. El script recorre las causas, guarda actuaciones.csv y descarga los adjuntos en expedientes_pjn/<numero> - <caratula>/.

Notas sobre el SCW

  • El SCW es una app JSF/Seam (no una SPA). Las URLs arrastran un parámetro de conversación cid que caduca, por eso el script re-localiza las filas en vez de guardar URLs.
  • Las causas penales y de familia no aparecen en la consulta pública.
  • Las esperas son por refresco de elementos (AJAX), no por cambio de URL.
⚙️ Ajustes necesarios. Los selectores marcados con # >>> AJUSTAR en scraper_pjn.py son genéricos. Inspeccioná el HTML real del sitio (F12 → Inspeccionar) y reemplazálos según la versión vigente del SCW.
ℹ️ Nota. El script necesita un entorno con interfaz gráfica porque abre el navegador visible para resolver el captcha. Se ejecuta en tu máquina, no en un servidor sin pantalla.
