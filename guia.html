📥 Cómo descargar los expedientes del Poder Judicial

Guía paso a paso — Sistema de Consulta Web (SCW) del Poder Judicial de la Nación · scw.pjn.gov.ar

Paso 1 — Instalar Python (una sola vez)

  1. Entrá a https://www.python.org/downloads/
  2. Apretá el botón amarillo grande “Download Python”.
  3. Abrí el archivo descargado.
  4. MUY IMPORTANTE: antes de instalar, tildá abajo la casilla “Add Python to PATH”.
  5. Apretá “Install Now” y esperá a que termine.

Paso 2 — Bajar el programa (una sola vez)

  1. Entrá a https://github.com/loekemeyer/Judicial
  2. Apretá el botón verde “Code” → “Download ZIP”.
  3. Descomprimí el ZIP (clic derecho → “Extraer todo”). Queda una carpeta llamada Judicial.

Paso 3 — Abrir la terminal en esa carpeta

  1. Entrá a la carpeta Judicial.
  2. En la barra de arriba (donde dice la ruta de la carpeta), escribí cmd y apretá Enter.
    Se abre una ventana negra: esa es la “terminal”.

Paso 4 — Preparar el programa (una sola vez)

Copiá y pegá esto en la ventana negra, uno por uno, apretando Enter y esperando que termine:

pip install playwright
playwright install chromium

Paso 5 — ¡Usarlo! (esto se hace cada vez)

Escribí esto y apretá Enter:

python scraper_pjn.py

Se abre solo un navegador con la web del Poder Judicial. Entonces:

  1. Escribí el apellido en el campo Parte.
  2. Resolvé el captcha.
  3. Apretá Buscar y esperá a ver la lista de causas.
  4. Volvé a la ventana negra y apretá Enter.
✅ El programa hace el resto solo: entra a cada causa, anota los movimientos y baja los PDFs en una carpeta llamada expedientes_pjn, dentro de la misma carpeta Judicial.

Si algo no anda — modo diagnóstico

La primera vez puede que el programa no encuentre bien la tabla de la web. Si pasa, escribí:

python scraper_pjn.py --debug
Eso crea una carpeta debug_scw. Enviá el archivo debug_scw/tablas.txt que aparece ahí y se ajusta el programa para que funcione perfecto con la web real. Es un único ajuste y queda andando.
